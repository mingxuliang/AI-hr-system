# AI 智能招聘管理系统 产品介绍

## 产品概述

AI 智能招聘管理系统是一款面向企业人力资源部门的全流程招聘管理平台，深度融合人工智能技术，实现从岗位发布、简历筛选、面试安排到 Offer 发放的全链路数字化管理。系统通过 AI 技术大幅提升招聘效率，降低人工筛选成本，帮助企业快速找到合适人才。

---

## 核心功能模块

### 一、岗位管理

岗位管理模块是企业招聘工作的起点，支持完整的岗位生命周期管理。

#### 1.1 岗位信息管理

- **创建岗位**：填写岗位名称、所属部门、工作地点、薪资范围、招聘人数等基本信息
- **岗位类型**：支持全职、兼职、合同、实习等多种用工形式
- **紧急程度**：设置岗位紧急级别（低/中/高/紧急），便于优先级排序
- **招聘负责人**：指派岗位负责人，明确责任归属

#### 1.2 AI 生成岗位描述（JD）

系统集成 AI 能力，可根据岗位名称自动生成专业的岗位职责和任职要求：

- 输入岗位名称后，点击「AI 生成 JD」按钮
- AI 自动生成结构化的岗位职责描述
- 自动生成详细的任职资格要求
- 支持人工编辑调整，确保内容准确

> **[截图位置：AI 生成 JD 对话框界面]**

#### 1.3 岗位发布与分享

- **发布岗位**：一键发布岗位，生成公开访问链接
- **复制链接**：快速复制岗位链接，分享至招聘渠道
- **下架岗位**：岗位招满后可随时下架
- **关联题库**：将题库与岗位关联，面试时自动参考

#### 1.4 招聘进度追踪

每个岗位实时展示招聘进度数据：

- 总简历数
- 待筛选数量
- 待面试数量
- 已入职数量
- 转化率统计

> **[截图位置：岗位列表页面，展示招聘进度条]**

### 二、简历管理与 AI 筛选

简历管理模块实现简历的全生命周期管理，AI 自动解析和初筛大幅提升筛选效率。

#### 2.1 简历上传

- **单个上传**：选择岗位后上传单个简历文件
- **批量上传**：支持一次上传多份简历（最多10份）
- **格式支持**：PDF、Word、Txt 等常见格式
- **公开投递**：候选人可通过岗位链接自主投递简历

> **[截图位置：简历上传对话框]**

#### 2.2 AI 自动解析

简历上传后，系统自动调用 AI 进行智能解析：

- **信息提取**：自动提取姓名、联系方式、教育背景、工作经历等关键信息
- **格式转换**：将简历内容转换为结构化的 Markdown 格式，便于阅读
- **解析状态**：实时显示解析中、解析成功、解析失败状态

#### 2.3 AI 初筛评分

系统根据岗位要求对简历进行智能匹配评分：

- **匹配度评分**：0-100 分制，量化简历与岗位的匹配程度
- **智能评价**：AI 生成详细的简历评价报告
- **筛选建议**：自动给出「通过」「淘汰」「待定」建议
- **人工复核**：AI 建议淘汰的简历可人工复核确认

> **[截图位置：简历详情页，展示 AI 初筛评价]**

#### 2.4 简历状态流转

系统支持完整的简历状态管理：

| 状态 | 说明 |
|------|------|
| 解析中 | 简历正在 AI 解析 |
| 待评审 | 解析完成，等待人工评审 |
| 待部门评审 | HR 初审通过，等待部门评审 |
| 待 HR 决策 | 部门评审通过，等待 HR 最终决策 |
| AI 建议淘汰 | AI 建议淘汰，等待人工确认 |
| 待面试 | 初审通过，可安排面试 |
| 面试通过 | 面试评估通过 |
| 面试未通过 | 面试评估未通过 |
| Offer 待确认 | 已发送 Offer，等待候选人确认 |
| 已接受 Offer | 候选人已接受 Offer |
| 已拒绝 Offer | 候选人已拒绝 Offer |
| 已录用 | 候选人已入职 |
| 已淘汰 | 简历已淘汰 |

#### 2.5 其他功能

- **重新解析**：解析失败或需要重新解析时可手动触发
- **恢复简历**：已淘汰的简历可恢复重新处理
- **简历搜索**：按候选人姓名、状态筛选简历

### 三、面试安排与评分

面试模块是系统的核心功能，支持多面试官协作、AI 辅助出题、录音转写等高级特性。

#### 3.1 安排面试

从简历列表点击「安排面试」进入面试安排界面：

- **面试轮次**：支持多轮面试，自动识别当前轮次
- **面试类型**：HR 面、技术面、主管面、CEO 面、综合面
- **面试形式**：现场面试、视频面试、电话面试
- **面试官选择**：支持选择多位面试官组成面试小组
- **时间地点**：设置面试时间和地点/会议链接
- **邮件通知**：自动生成面试通知邮件，可编辑后发送

> **[截图位置：安排面试对话框]**

#### 3.2 AI 生成面试题

系统根据简历内容和面试类型自动生成定制化面试题：

- **智能出题**：基于候选人简历经历生成针对性问题
- **题库参考**：可选择题库作为参考，生成更精准的题目
- **题目数量**：自定义生成题目数量（1-20 道）
- **跳过生成**：可选择跳过 AI 出题，手动添加题目

每道面试题包含：

- 问题标题和内容
- 简历关联（关联的简历经历）
- 参考答案
- 评分标准
- 追问方向
- 难度等级（初级/中级/高级）
- 问题类型（技术/项目/行为）

> **[截图位置：面试评分页面，展示面试题目]**

#### 3.3 面试评分界面

面试官进入评分界面后，可进行以下操作：

**左侧 - 简历预览**
- 实时预览候选人简历
- 支持 PDF 在线预览
- 可下载简历文件

**右侧 - 面试评分**
- 题目导航：快速跳转到任意题目
- 评分输入：0-10 分制打分
- 评语填写：针对每题填写评语
- 题目编辑：可编辑、删除、新增题目

> **[截图位置：面试评分双栏界面]**

#### 3.4 面试录音与实时转写

系统支持整场面试录音，并提供实时语音转写：

- **开始录音**：首位面试官可开启录音
- **实时转写**：录音时实时显示语音识别文字
- **说话人区分**：AI 自动区分不同说话人
- **录音上传**：录音结束后手动或自动上传
- **AI 分析**：上传后 AI 结合录音内容生成综合评价

> **[截图位置：录音状态栏，展示实时转写文字]**

#### 3.5 面试计时

- 点击「开始面试」按钮启动计时
- 实时显示面试已进行时间
- 面试开始后才能提交评分

#### 3.6 多面试官协作

支持多位面试官同时参与评分：

- **独立评分**：每位面试官独立提交评分
- **状态同步**：实时显示各面试官提交状态
- **等待机制**：等待所有面试官提交后进入 AI 分析
- **综合评价**：AI 综合所有面试官评分生成评价

#### 3.7 面试结果确认

面试评分提交后，AI 自动生成综合评价：

- **AI 建议**：给出录用/淘汰/待定/进入下一轮建议
- **HR 确认**：HR 可接受或修改 AI 建议
- **结果修改**：支持后续修改面试结果

#### 3.8 面试报告导出

支持导出完整的面试评估报告：

- **Markdown 格式**：便于编辑和存档
- **PDF 格式**：正式报告格式
- **报告内容**：包含简历初审评价、面试评分详情、AI 综合评价

> **[截图位置：面试结果页面]**

### 四、Offer 管理

Offer 管理模块实现从创建到确认的完整流程管理。

#### 4.1 创建 Offer

- **选择候选人**：从通过面试的候选人中选择
- **自动填充**：自动填充候选人信息和岗位信息
- **Offer 模板**：可应用预设模板快速填充
- **详细信息**：薪资、入职日期、试用期、福利待遇等

#### 4.2 Offer 模板管理

- 创建岗位专属或通用 Offer 模板
- 设置默认模板自动应用
- 模板包含薪资结构、福利、特殊条款等

#### 4.3 Offer 流程

| 状态 | 说明 |
|------|------|
| 草稿 | Offer 创建中，可编辑 |
| 待发送 | 准备发送 |
| 已发送 | 已发送给候选人 |
| 已接受 | 候选人已接受 |
| 已拒绝 | 候选人已拒绝 |
| 已过期 | 超过有效期 |
| 已撤回 | HR 主动撤回 |

#### 4.4 候选人确认页面

候选人通过链接查看 Offer 详情：

- 完整的 Offer 信息展示
- 在线接受或拒绝
- 填写拒绝原因反馈
- 确认入职日期

> **[截图位置：候选人 Offer 确认页面]**

#### 4.5 其他功能

- **重新打开**：已结束的 Offer 可重新打开
- **删除 Offer**：支持删除任意状态的 Offer
- **统计面板**：Offer 总数、接受率等统计数据

### 五、题库管理

题库管理模块用于维护面试题目资源库。

#### 5.1 题库上传

- **题库名称**：为题库命名
- **关联岗位**：关联到具体岗位
- **分类标签**：技术/管理/人力资源/其他
- **难度等级**：初级/中级/高级
- **自定义标签**：添加技术栈等标签
- **文件上传**：支持 PDF、Word、Txt、Markdown 格式

#### 5.2 文件预览

- PDF 文件在线预览
- Word 文档在线预览
- Markdown 渲染预览
- 文本文件直接显示

> **[截图位置：题库详情抽屉，展示文件预览]**

#### 5.3 题库应用

- 面试出题时参考题库内容
- AI 根据题库生成更精准的面试题
- 岗位关联题库便于管理

- 平均匹配分数

可视化展示招聘各阶段转化：

```
简历接收 → 初筛通过 → 面试安排 → 面试通过 → Offer 发送 → 入职
```

- 各阶段人数统计
- 转化率计算
- 整体转化率展示

> **[截图位置：招聘漏斗图表]**

#### 7.3 时间趋势分析

- 可选时间范围：7天/14天/30天/60天/90天

#### 7.5 面试官分析

#### 7.6 最新动态

- 简历投递
